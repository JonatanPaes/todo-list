# Como Executar?

> Clone o Repositório:

Node version 18.16.0

Clone

```bash
git clone https://github.com/JonatanPaes/todo-list.git
```

Para executar o projeto

```bash
cd todo-list
```

Instale as dependências do projeto

```bash
yarn install
```

# Web
Para executar o projeto

```sh
yarn start
```
